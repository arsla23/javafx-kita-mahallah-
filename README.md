KITA Mahallah Management System (JavaFX UI)

This project implements a JavaFX-based graphical user interface for the KITA Mahallah Management System. The focus of the project is on UI design and layout, not backend or database functionality.

The Admin Dashboard was developed using JavaFX layout containers such as BorderPane, VBox, GridPane, and ScrollPane to organize the interface into a sidebar, main content area, and notification panel. The design reflects the roles and responsibilities defined in the UML class diagram.

The project demonstrates the process of translating UML diagrams into working JavaFX components, applying basic styling, and structuring a desktop application interface.

Technologies used: Java, JavaFX, Maven
Scope: Frontend UI implementation only
